# ORCA Hand Feetech Version

这是一个 ORCA Hand 17 自由度灵巧手的低成本复现版本。我在原项目基础上尝试使用飞特 STS3215 V12 舵机替代原方案中的舵机，并整理了一套当前阶段可用于硬件搭建的资料。

当前仓库还只是一个非常简易的初版，主要由我和 Codex 一起整理完成。你可以先参考这里的文档、物料清单、3D 打印文件和走线图片来搭建硬件部分。后面一两周我会继续快速迭代这个 repo，逐步补充更清晰的装配说明、踩坑记录、调试流程和更多实拍图片。

## 当前内容

- `docs/`：中文教程初版和几个简单检查表。
- `hardware/物料清单.xlsx`：当前版本用到的主要物料。
- `hardware/3d_print/`：官方 3D 模型和适配飞特舵机的改版模型。
- `hardware/走线图片/`：当前整理出的走线参考图片。

## 当前状态

目前这个 repo 更适合作为“硬件搭建资料包”使用，还不是一个完整、成熟、一步到位的开源项目。

如果你正在复现，可以先从 `docs/orcahand_sts3215_tutorial.md` 开始看，然后结合 `hardware/` 目录中的资料逐步搭建。后续我会继续完善教程结构、图片标注、舵机调试记录和常见问题。
