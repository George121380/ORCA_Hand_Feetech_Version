## 下单前确认

- [ ] 已确认制作左手还是右手。
- [ ] 已确认打印材料。
- [ ] 已确认每个文件的打印数量。
- [ ] 已确认是否需要打印皮肤件，或改用硅胶倒模。
- [ ] 已确认服务商能保证小孔、薄壁和轴承孔精度。

## 到货后处理

- [ ] 清理支撑和毛刺。
- [ ] 清理走线孔。
- [ ] 检查轴承孔。
- [ ] 检查销钉孔。
- [ ] 检查舵机仓能否装入 STS3215。
- [ ] 检查棘轮机构能否转动和定位。
- [ ] 使用导管试穿主要路径。
- [ ] 对偏小孔位轻微扩孔。

## 装配前确认

- [ ] 手指关节能自由转动。
- [ ] 手掌和手指安装位对齐。
- [ ] 塔尖和舵机仓能合体。
- [ ] 舵机线束有可用出线空间。
- [ ] 所有需要后处理的孔位已经处理完成。
