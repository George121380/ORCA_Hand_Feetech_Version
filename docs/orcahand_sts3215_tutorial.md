# ORCA Hand STS3215 V12 低成本复现教程

本文是一份面向初学者的复现教程初版，目标是帮助你从零完成一个使用飞特 STS3215 V12 舵机的 ORCA Hand 低成本版本。

当前教程优先覆盖硬件复现闭环：采购、3D 打印、机械装配、穿线、舵机编号、飞特调试软件测试。后续可以继续补充实拍图、视频链接、控制代码和更完整的动作调参方法。

> 说明：原始资料中有些地方写作 `ST3215`，本文统一写作 `STS3215 V12`。采购时请以你实际购买的舵机型号为准。

## 1. 项目介绍

ORCA Hand 是一个 17 自由度灵巧手项目。本版本保留 ORCA Hand 的主要结构思路，并将舵机方案改成飞特 STS3215 V12，从而降低复现成本。

这只手主要由以下部分组成：

- 手掌和五指骨架：负责形成手的机械结构。
- 皮肤或软质接触件：提升抓握时的接触效果。
- 腱绳系统：使用 PE 路亚线、特氟龙管和棘轮机构传递舵机拉力。
- 舵机仓和塔尖：集中安装 17 个舵机，并将拉线引出到手部。
- 飞特舵机调试系统：用于设置舵机 ID、检查舵机方向、测试基础动作。

这份教程默认你要复现的是右手版本。如果你制作左手版本，3D 打印文件、走线方向和左右筋命名都需要相应镜像确认。

## 2. 复现前准备

### 2.1 适合人群

你不需要有完整机器人项目经验，但最好具备以下基础：

- 能看懂简单机械装配关系。
- 会使用 3D 打印服务或切片软件。
- 能在 Windows 上安装驱动和运行调试软件。
- 愿意反复调线、拆装和微调。

如果你是第一次做类似项目，建议先把目标定为“单指能稳定弯曲和回弹”，不要一开始就追求整手复杂动作。

### 2.2 推荐复现顺序

1. 先采购舵机、驱动板、电源、线材、轴承、螺丝和销钉。
2. 同时下单 3D 打印件。
3. 打印件到手后先清理毛刺和试装轴承。
4. 单独给 17 个舵机设置 ID，并贴好编号标签。
5. 装配手掌、手指、塔尖和舵机仓。
6. 先穿特氟龙管，再穿 PE 线。
7. 用棘轮机构逐根调线。
8. 使用飞特调试软件做单舵机、单指、整手测试。

### 3.3 采购建议

- 舵机最好一次买齐 17 个，避免批次差异导致调试体验不同。
- 螺丝、销钉、端子、特氟龙管建议多买，因为初次装配很容易剪短、压坏或丢失。

- 3D 打印件下单前先确认左右手版本、材料、数量和是否需要后处理。

### 4.3 打印和后处理建议

- 手掌骨架建议使用强度较高、尺寸稳定的材料。

## 5. 机械装配

机械装配建议按模块进行，不要一开始就把所有线都穿好。每个模块先确认能自由转动，再进入下一步。

### 5.1 打印件检查

1. 按文件名和数量清点所有打印件。
2. 检查关节孔、轴承孔、走线孔是否堵塞。
3. 清理支撑残留、毛刺和树脂残留。
4. 用对应轴承、销钉、螺丝逐个试装。
5. 对偏紧孔位做轻微扩孔，扩孔后再次试装。

检查标准：

- 轴承能压入或轻微过盈安装，不应松到自行脱落。
- 销钉能插入并形成转轴，关节能转动。
- 腱绳和特氟龙管能顺利穿过路径孔。

### 5.2 手指装配

建议按食指、中指、无名指、小指、拇指的顺序逐根装配。

每根手指的基本步骤：

1. 将指尖、中节、根部结构件按模型方向摆好。
2. 先装轴承，再插入对应销钉。
3. 检查每个关节是否能在预期方向转动。
4. 暂时不要剪短腱绳，先预留足够长度。
5. 先不装皮肤，等动作测试完成后再考虑安装。

装配时要注意背筋和腹筋的路径：

- 背筋通常负责让手指向伸展方向运动。
- 腹筋通常负责让手指向弯曲方向运动。
- 根部左右筋负责左右摆动或侧向动作。

实际方向以第 7 节的舵机对应表和第一次小角度测试为准。

### 5.3 手掌和腕部装配

1. 先装手掌骨架和腕部结构。
2. 检查手指根部安装位是否平整。
3. 逐个安装五根手指。
4. 确认每根手指能在不接线的情况下自然活动。
5. 将特氟龙管沿手掌内部路径预放进去。

建议在这个阶段拍照记录每根线和每根管的位置。后续如果动作异常，照片能帮助快速回退。

### 5.4 塔尖和舵机仓装配

1. 将两个舵机仓打印件先空载试装。
2. 将 17 个舵机按预定 ID 贴标签。
3. 将舵机放入舵机仓，确认出线方向和舵机盘方向。
4. 安装塔尖主体、塔尖电机齿轮、卡扣、转接件和轴承盖。
5. 安装 16 套棘轮机构。

舵机仓空间比较紧张，出线方向要尽早规划。推荐做法：

- 先把舵机线朝同一侧整理。
- 每 3-5 个舵机一组，用临时扎带轻轻固定。
- 不要把线束压在棘轮活动范围内。
- 确认所有舵机都能单独拆出后，再做最终固定。

### 5.5 棘轮机构装配

棘轮用于调整腱绳张力。初次装配时不要调得太紧。

基本步骤：

1. 将棘轮齿轮、棘轮连接、棘轮盖配成一组。
2. 确认棘轮能转动，并能在齿位上停住。
3. 将对应腱绳绕入棘轮或连接到舵机盘。
4. 每根线只预紧到“刚好不松垂”。
5. 等所有线都接好后，再逐根微调。

常见错误：

- 一开始拉得太紧，导致舵机一上电就堵转。
- 棘轮方向装反，越调越松。
- 线从棘轮边缘滑出，动作时突然失去张力。

## 6. 穿线与走线

走线参考图片位于：

- `../hardware/走线图片/`

### 6.1 穿线材料

| 材料 | 用途 |
| --- | --- |
| PE 路亚线 6 号 | 作为主要腱绳 |
| 特氟龙管 | 作为腱绳导管，降低摩擦并约束路径 |
| 0.3mm / 0.4mm 钢丝 | 辅助穿线或辅助结构 |
| 棘轮机构 | 调整腱绳张力 |
| 舵机盘 | 固定另一侧腱绳或形成拉线输出 |

### 6.2 穿线原则

1. 先穿特氟龙管，再穿 PE 线。
2. 每根线先留出 10-20cm 余量。
3. 同一根手指的背筋、腹筋不要交叉。
4. 经过关节的线要贴近设计路径，避免摩擦打印件边缘。
5. 每穿完一根线，就用标签记录它对应的舵机 ID 和筋名。
6. 不要在所有动作测试完成前剪到最终长度。

### 6.3 推荐穿线顺序

1. 食指：指尖背筋、指尖腹筋、中背筋、中腹筋、根左筋、根右筋。
2. 中指：指尖背筋、指尖腹筋、中背筋、中腹筋、根左筋、根右筋。
3. 无名指：指尖背筋、指尖腹筋、中背筋、中腹筋、根左筋、根右筋。
4. 小指：指尖背筋、指尖腹筋、中背筋、中腹筋、根左筋、根右筋。
5. 拇指：尖背筋、尖腹筋、中背筋、中腹筋、根前筋、根后筋、根上筋、根下筋。
6. 手腕：最后接入，避免前面装配时反复拉扯。

### 6.4 张力调整

张力调整要慢，建议按以下方式：

1. 所有舵机先回到中立位置。
2. 每根线先调到不明显松垂。
3. 手动活动对应手指，确认线不会卡住。
4. 用调试软件让对应舵机小角度运动。
5. 如果动作太小，再收紧一点。
6. 如果一上电就有明显拉扯或舵机声音异常，先放松该线。

## 8. 飞特调试软件配置

飞特调试软件、驱动和手册建议从飞特官方渠道或购买调试板随附资料中获取。建议先阅读：

- `先看这里-入手教程20250301.pdf`
- `SCServo-Debug使用手册1.9.8.pdf`
- `FE-URT-1调试板/URT-1使用说明.pdf`
- `STS3215-C001串型规格书(增加运动曲线图)-20230624.pdf`

### 8.1 安装驱动和软件

1. 准备一台 Windows 电脑。
2. 安装 CH340 驱动。
3. 插入 FE-URT-1 或对应舵机驱动板。
4. 在设备管理器中确认串口号。
5. 打开飞特调试软件，例如 FD 1.9.8.4 版本。

### 8.2 连接舵机

首次设置 ID 时，一次只连接一个新舵机。

基本连接顺序：

1. 电脑 USB 连接调试板。
2. 调试板连接舵机总线。
3. 外部电源连接舵机供电端。
4. 确认电源电压和极性。
5. 打开调试软件并选择正确串口。
6. 搜索舵机。

如果软件无法识别舵机：

- 检查 CH340 驱动是否安装成功。
- 检查串口号是否选择正确。
- 检查舵机供电是否打开。
- 检查总线插头方向是否正确。
- 只保留一个舵机重新搜索。

### 8.3 设置 17 个舵机 ID

推荐流程：

1. 连接第一个舵机。
2. 搜索到舵机后，将 ID 修改为 `1`。
3. 断电，贴上 `ID 1` 标签。
4. 记录该舵机将对应的筋名。
5. 换下一个舵机，重复以上步骤，直到 `ID 17`。

不要在多个未编号舵机同时连接的情况下修改 ID，否则容易把多个舵机改成同一个 ID。

### 8.4 中立位置和方向检查

每个舵机装入结构前，建议先做空载检查：

1. 让舵机回到中立位置。
2. 在舵机盘上做一个方向标记。
3. 用调试软件给一个很小的位置变化。
4. 记录顺时针和逆时针对应的实际转动方向。
5. 对照第 7 节表格，确认它拉紧的是正确那根线。

如果方向和预期相反，优先通过换线位置、舵机盘安装角度或控制方向来调整。不要强行用过大角度抵消装配问题。

## 9. 基础动作测试

基础测试建议分四轮进行。

### 9.1 第一轮：舵机空载测试

目标：确认所有舵机 ID 正确，能被调试软件单独控制。

步骤：

1. 只连接舵机，不连接腱绳。
2. 逐个选择 ID 1-17。
3. 每个舵机只做小角度运动。
4. 记录是否能响应、是否方向正常、是否有异常声音。

通过标准：

- 17 个舵机都能被单独识别。
- 每个舵机 ID 与标签一致。
- 每个舵机都能顺畅转动。

### 9.2 第二轮：单根腱绳测试

目标：确认每根线都接到正确位置。

步骤：

1. 选择一个舵机 ID。
2. 只轻微拉紧它对应的一组线。
3. 小角度控制舵机。
4. 观察对应手指或关节是否按预期运动。
5. 立即记录结果。

如果发现拉错线，不要继续调角度，先断电并重新整理该线。

### 9.3 第三轮：单指测试

目标：确认单根手指能完成基础弯曲、伸展或侧摆。

推荐顺序：

1. 食指。
2. 中指。
3. 无名指。
4. 小指。
5. 拇指。

每根手指测试时，先测指尖，再测中节，最后测根部。根部左右摆动最容易受线长和导管摩擦影响，需要单独检查。

### 9.4 第四轮：整手测试

目标：确认多根手指同时动作时，线束不会互相干涉。

建议动作：

1. 五指轻微张开。
2. 食指轻微弯曲。
3. 中指轻微弯曲。
4. 五指一起轻微弯曲。
5. 拇指对掌方向轻微动作。
6. 手腕小范围动作。

整手测试阶段不要追求握紧。先确保每个动作都可重复、不卡线、不明显偏斜。

## 10. 常见问题与改进建议

以下内容来自当前复现过程中的改进记录，并补充了初学者排查方法。

### 10.1 穿线孔偏小

现象：

- 特氟龙管穿不过。
- PE 线通过时摩擦明显。
- 手指回弹不顺畅。

处理：

- 装配前先用导管逐个试穿。
- 使用手钻或扩孔工具轻微扩孔。
- 扩孔后清理毛刺，避免割线。
- 如果你有建模能力，可以在源文件中把孔径适当放大。

### 10.2 塔尖舵机固定不方便

现象：

- 舵机装入后不容易拆。
- 出线方向难整理。
- 固定件安装时容易和线束干涉。

处理：

- 先空载试装舵机仓，不要直接装线。
- 先规划出线方向，再固定舵机。
- 每个舵机线束贴编号。
- 后续可以优化塔尖模型，在内部增加更方便拆装的卡槽。

### 10.3 棘轮步距偏大

现象：

- 转一格太松，再转一格又太紧。
- 张力很难调到刚好合适。

处理：

- 初版先接受较粗的调节精度，用线结位置和舵机盘角度辅助微调。
- 调试时保持小角度动作，避免因为张力过大导致结构卡住。
- 后续可以重新设计更小步距的棘轮结构。

### 10.4 舵机走线空间紧张

现象：

- 舵机线束挤压棘轮。
- 舵机仓合不上。
- 某些舵机不好拆装。

处理：

- 舵机安装前先模拟线束走向。
- 线束按 ID 分组固定，不要全部堆在一个角落。
- 检查每条线束是否避开活动件。
- 后续可以重新设计出线位置和舵机仓内部空间。
