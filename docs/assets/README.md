# 教程图片素材目录

这个目录用于放教程中筛选、重命名和标注后的图片。

当前原始走线图片位于：

- `../../hardware/走线图片/`

建议后续整理时使用稳定命名，例如：

- `wiring_index_finger_01.jpg`
- `wiring_middle_finger_01.jpg`
- `wiring_thumb_01.jpg`
- `servo_box_layout_01.jpg`
- `feetech_debug_id_setup_01.jpg`

整理原则：

- 一张图片只表达一个装配或调试动作。
- 文件名使用英文和数字，避免后续发布平台处理中文路径时出错。
- 图片中尽量加入箭头、编号、舵机 ID 或筋名标注。
- 替换主教程中的原始图片引用时，优先使用本目录图片。
